# Dogaria

犬のおしっこ記録から縄張りを可視化する、スマホ向けの無料Webアプリです。

## できること
- 犬名入力
- 写真アップロード
- EXIF優先 / GPSフォールバックで位置取得
- 15cm定規を基準にしたおしっこ高さ推定
- localStorage保存
- OpenStreetMap上での記録表示
- 犬ごとの縄張り境界の近似表示

## 公開
GitHub Pages で公開します。

## 技術
- HTML
- CSS
- JavaScript
- Leaflet
- OpenStreetMap
- EXIF.js

## 注意
- 記録は端末内のブラウザにのみ保存されます
- サーバーには送信しません
